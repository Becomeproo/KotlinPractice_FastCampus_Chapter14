* RecyclerView 사용
* View Binding 사용

### 중고 거래 앱
* Firebase Authentication 기능을 사용하여 로그인 회원가입 기능 구현
* 회원 기반으로 중고거래 아이템 등록 가능
* 아이템 등록 시 사진 업로드를 위해 Firebase Storage를 사용
* 회원 기반으로 채팅화면 구현
* Fragment를 사용하여 하단 탭 화면 구조를 구현
* FloatingActioinButton 사용
